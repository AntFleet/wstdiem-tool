# WSTDIEM Loop Manager

WSTDIEM Loop Manager is an operator toolkit for monitoring wstDIEM loop health and evaluating safety-critical actions before any live execution. It focuses on evidence, readiness checks, and conservative simulations so operators can reason about positions without exposing sensitive runbook details.

**Operator on-ramp:** [docs/deployment/operator-runbook.md](docs/deployment/operator-runbook.md) — sizing (candidate/marginal/blocked), monitor exit-code gating, liquidation readout, exit rehearsal, drained-pool fail-closed, and intelligence commands (`capacity` / `brief` / `demand` / `basis`).
